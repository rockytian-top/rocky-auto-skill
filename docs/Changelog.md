# 更新日志 | Changelog

All notable changes to this project will be documented in this file.

---

## [v2.9.0] - 2026-04-19

### 修复 | Bug Fixes

- **路径独立化问题**: 修复了插件在不同 OpenClaw 安装路径下的兼容性问题，确保 `.auto-skill` 目录始终在正确的 `~/.openclaw/` 下创建
- **共享技能目录复制逻辑**: 改进了从 `shared-skills` 目录自动复制脚本的逻辑，支持更多目录变体

---

## [v2.7.0] - 2026-04-18

### 新增 | Features

- **完整中英文文档**: 重写 README.md，提供完整的中英文双语文档
- **功能对比表**: 新增 OpenClaw 无插件 vs 安装插件 vs Hermes Agent 的详细对比
- **三级晋升机制文档化**: 完整记录 L1 → L2 → L3 晋升流程和条件

### 优化 | Improvements

- **README 排版优化**: 使用 Unicode 表格和更清晰的分区
- **目录结构说明**: 完善了安装后的目录结构说明

---

## [v2.6.0] - 2026-04-17

### 新增 | Features

- **BM25 + 向量混合搜索**: 实现混合搜索引擎，支持精确关键词匹配 + 语义相似度
- **Shell 包装器**: 提供 `autoskill-search` Shell 脚本，方便命令行调用

### 优化 | Improvements

- **搜索精度提升**: 通过混合搜索提升匹配准确性
- **向量嵌入存储**: 卡片支持存储向量嵌入，加速相似度计算

---

## [v2.5.0] - 2026-04-16

### 新增 | Features

- **自动晋升机制**: 实现 L1 → L2 → L3 渐进晋升
- **成功率追踪**: 每次脚本执行后自动更新统计信息

### 修复 | Bug Fixes

- **缓存失效问题**: 修复了缓存 TTL 计算错误导致的频繁刷新

---

## [v2.4.0] - 2026-04-15

### 新增 | Features

- **L3 自动执行**: 成功率 ≥90% 的技能自动执行，无需人工确认
- **执行结果注入**: 脚本执行结果自动注入 AI 上下文

### 优化 | Improvements

- **执行流程优化**: 减少不必要的日志输出，静默执行

---

## [v2.3.0] - 2026-04-14

### 新增 | Features

- **L2 确认执行**: 进阶技能需人工确认后执行
- **hit 计数**: 每次搜索命中自动计数

### 修复 | Bug Fixes

- **并发安全问题**: 修复了多并发场景下的缓存竞态问题

---

## [v2.2.0] - 2026-04-13

### 新增 | Features

- **错误检测**: 自动从 AI 响应中提取错误信息
- **经验卡片 YAML 格式**: 统一使用 YAML 格式存储经验卡片

### 优化 | Improvements

- **搜索触发优化**: 仅在消息 ≥3 字符（中文）时触发搜索

## [v2.1.0] - 2026-04-12

### 新增 | Features

- **缓存机制**: 5 分钟 L3 技能缓存，减少 IO

- **模板脚本发现**: 自动发现并加载 `skills/` 目录下的模板脚本

## [v2.0.0] - 2026-04-11

## 版本规范 | Versioning

项目使用 [语义化版本](https://semver.org/lang/zh-CN/)：

```
主版本.次版本.修订号
MAJOR.MINOR.PATCH

- MAJOR: 不兼容的 API 变更
- MINOR: 向后兼容的功能新增
- PATCH: 向后兼容的问题修复
```

## 如何贡献更新日志 | How to Add Changelog

1. 在 `CHANGELOG.md` 顶部添加新版本
2. 使用以下前缀：
   - `### 新增 | Features`
   - `### 优化 | Improvements`
   - `### 修复 | Bug Fixes`
   - `### 重构 | Refactoring`
3. commit 后自动包含在发布说明中
