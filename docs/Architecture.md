# 架构说明 | Architecture

## 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                     OpenClaw Gateway                        │
│  ┌─────────────────────────────────────────────────────┐  │
│  │              rocky-auto-skill Plugin                 │  │
│  │  ┌─────────────────────────────────────────────────┐  │  │
│  │  │            before_agent_start Hook              │  │  │
│  │  │  1. 错误/用户消息检测                          │  │  │
│  │  │  2. 经验搜索 (BM25 + 向量)                    │  │  │
│  │  │  3. 匹配技能 → 模型判断是否执行               │  │  │
│  │  │  4. 脚本执行 (autoExecuteScript)               │  │  │
│  │  │  5. 结果注入 (prependContext)                 │  │  │
│  │  │  6. 模型驱动的工作流分析                       │  │  │
│  │  └─────────────────────────────────────────────────┘  │  │
│  └─────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
```
